# 📈 Stock Portfolio Tracker - Python

## 📌 Project Overview

The **Stock Portfolio Tracker** is a simple Python-based application developed as part of the **CodeAlpha Python Programming Internship**.

This project allows users to track stock investments by entering stock names and quantities. The application uses a predefined dictionary of stock prices, calculates investment values, and displays the total portfolio investment amount.

Additionally, the project stores the portfolio details in a text file for future reference.

This project demonstrates the use of Python dictionaries, loops, arithmetic operations, user input handling, and file handling.

---

## 🎯 Objective

The main objective of this project is to:

- Build a stock portfolio tracker using Python
- Accept stock names and quantities from users
- Calculate investment values
- Display the total investment amount
- Save portfolio details into a text file

---

## ✨ Features

✔ User-friendly console interface  
✔ Hardcoded stock prices using dictionary  
✔ User input for stock names and quantities  
✔ Calculates individual stock investment values  
✔ Calculates total investment amount  
✔ Displays portfolio summary  
✔ Saves portfolio details into a `.txt` file  
✔ Handles unavailable stock names
